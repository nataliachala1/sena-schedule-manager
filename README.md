# System Modules Definition

El sistema divide en modulos funcionales según sus responsabilidades. 
Cada módulo agrupa entidades y operaciones relacionadasm para mejorr la cohesioón, la escalabilidad y la mantenibilidad 

## Modulo 1 Resource Management Module

### Entities
- room
- instructor

Descripción: Este módulo gestiona los recursos físicos y humanos del sistema.

Responsibilities: 

- Crear, actualizar y desactivar salas
- Gestionar la capacidad, ubicación y equipamiento de las salas
- Registrar y gestionar instructores
- Gestionar la información personal y profesional de los instructores
- Mantener la disponibilidad de los recursos
